## 🚀 How to Install

1. **Download** the desired `.json` file from this folder.
2. Open **NotePlan**.
3. Go to `Settings` > `Themes`.
4. Click on **"Open Themes Folder"**.
5. **Move** the downloaded `.json` file into that folder.
6. Select your new theme in the NotePlan settings.

---

## 🛠 Features
- Optimized for **Avenir Next** typography.
- Custom colors for `working-on`, `flagged`, and `important` tags.
- Dedicated styling for "Insight" (`--`) and "Question" (`?`) lines.
